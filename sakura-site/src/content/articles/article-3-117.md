## 核心步骤详解

### 步骤一：获取专属订阅链接

无论你用的是哪款客户端，核心都是要先拥有一个支持协议的订阅链接。

> [!WARNING] 
> **隐私警告**：千万不要把你的订阅链接发给任何人！这就相当于你的网银密码，一旦泄露，别人就可以疯狂消耗你的流量，甚至导致你的账号被风控封禁。

### 步骤二：导入与更新配置

通常只需要点击客户端面板上的 `Profiles` 或 `订阅` 按钮，将你的链接粘贴进去并点击“更新”或“Download”。

```yaml
# 一段典型的配置切片示例，仅供参考，切勿直接复制
proxies:
  - name: "HK-Premium-01"
    type: ss
    server: node.example.com
    port: 443
    cipher: chacha20-ietf-poly1305
```

### 步骤三：开启系统代理

完成导入后，你需要找到类似 **"System Proxy"（系统代理）** 或 **"Tun Mode"（虚拟网卡模式）** 的开关。

## 常见问题排查 (Troubleshooting)

| 症状表现 | 可能的原因 | 解决办法 |
| :--- | :--- | :--- |
| **能登 Telegram 但网页打不开** | DNS 污染 / 漏选系统代理 | 检查 System Proxy 是否勾选，开启 Tun 模式 |
| **所有节点全部超时 (Timeout)** | 订阅已过期 / 本地网络故障 | 尝试用手机热点测试，或前往官网续费 |
| **速度极慢，一直在转圈** | 选到了晚高峰拥堵的直连节点 | 手动在节点列表切换到带有 IPLC 或 BGP 字眼的专线节点 |

> [!TIP] 
> 💡 **终极杀手锏**：如果按照上述步骤还是不行，请尝试使用“重启大法”，或者在客户端的设置中点击 `Clear Cache` 清除缓存后再次更新订阅。
