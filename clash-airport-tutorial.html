Clash 机场使用教程：快速导入订阅，一键选择节点

更新时间：2026-05-17 作者：柳如烟

Clash 是目前各平台上最强大、最流行的代理客户端之一。由于其界面专业、功能丰富，许多新手第一次打开时会感到无从下手。其实，只要掌握了核心步骤，使用 Clash 非常简单。本文将手把手教你如何快速导入机场订阅并一键连接。

一、准备工作

在开始之前，你需要准备好两样东西：

1. 一个已经购买了套餐的机场：登录你的机场后台，找到“一键订阅”或“订阅链接”区域，复制你的 Clash 专属订阅链接。

2. 下载并安装 Clash 客户端：Windows 用户推荐使用 Clash Verge，Mac 用户推荐使用 ClashX 或 Clash Verge。请务必从 GitHub 官方仓库或靠谱的渠道下载，避免下载到带有木马的修改版。

二、三步完成订阅导入

第一步：打开订阅设置
打开你的 Clash 客户端，在左侧导航栏中找到“Profiles（配置）”或“Subscriptions（订阅）”选项卡。

第二步：粘贴并下载订阅
点击“新建”或在地址栏中，粘贴你刚刚从机场后台复制的订阅链接。点击“Download（下载）”或“Import（导入）”。此时客户端会向机场服务器请求节点信息，通常几秒钟内就会出现一个新的配置文件。

第三步：选中配置文件
在列表中看到下载好的配置文件后，点击选中它。被选中的文件通常会变成蓝色或有一个打勾的标志。这就说明客户端已经加载了该机场的节点列表。

三、选择节点并开启代理

进入客户端的“Proxies（代理）”界面。在这里，你会看到你的机场提供的所有节点（如香港、日本、美国等）。

在顶部通常会有一个“Rule（规则）”或“Global（全局）”模式的选择：

  • Rule（规则模式，强烈推荐）：智能分流，国内网站直连，国外网站走节点。
  • Global（全局模式）：所有流量强制走代理节点。
  • Direct（直连模式）：所有流量都不走代理（相当于关闭翻墙）。

确保选择“Rule”模式。然后，在下方的节点列表中，点击一个延迟较低的节点（如香港 01）。

最后，回到客户端的主界面或常规设置中，打开“System Proxy（系统代理）”开关。恭喜你，现在你可以自由访问海外网站了！

四、常见问题排查

如果开启后无法上网，请检查：系统时间是否准确（代理协议对时间极度敏感）；是否同时开启了其他 VPN 软件导致冲突；尝试点击节点列表右上角的“闪电”图标测试延迟，排除个别节点故障。
